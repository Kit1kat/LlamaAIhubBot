# LlamaAIhubBot 🦙🤖

Telegram бот на основе Llama AI (через Fireworks API) с поддержкой памяти разговоров.

## 🚀 Возможности

- **💬 Диалоги с AI**: Общение с Llama через Fireworks API
- **🧠 Память разговоров**: Бот запоминает важные моменты из прошлых диалогов
- **📝 Управление памятью**: Просмотр и очистка сохраненной информации
- **⚡ Асинхронный**: Быстрые и неблокирующие операции

## 📋 Требования

- Python 3.8+
- Telegram Bot Token (от @BotFather)
- Fireworks API Key (получить на https://www.fireworks.ai/)

## 🔧 Установка

### 1. Клонируем репозиторий
```bash
git clone https://github.com/Kit1kat/LlamaAIhubBot.git
cd LlamaAIhubBot
```

### 2. Устанавливаем зависимости
```bash
pip install -r requirements.txt
```

### 3. Создаём переменные окружения
```bash
export TG_TOKEN="ваш_telegram_token"
export FIREWORKS_KEY="ваш_fireworks_api_key"
```

Или создайте файл `.env`:
```
TG_TOKEN=your_token_here
FIREWORKS_KEY=your_key_here
```

### 4. Запускаем бота
```bash
python main.py
```

## 🌐 Развертывание

### На Heroku
1. Создайте аккаунт на Heroku
2. Установите Heroku CLI
3. Выполните:
```bash
heroku create your-app-name
heroku config:set TG_TOKEN="your_token" FIREWORKS_KEY="your_key"
git push heroku main
```

### На других сервисах
Бот использует Flask для keep_alive функции, что позволяет развертывать на:
- Replit
- Railway
- Render
- Другие облачные платформы

## 🎮 Как пользоваться

1. Найдите бота в Telegram и нажмите `/start`
2. Выберите опцию "Новый диалог" чтобы начать разговор
3. Пишите сообщения - бот будет отвечать на основе Llama AI
4. Используйте кнопку "Память" чтобы увидеть сохраненные разговоры
5. Нажмите "Сбросить память" чтобы очистить историю

## 🔑 Команды

- `/start` - Начало работы с ботом

## ⚙️ Команды через кнопки

- 💬 **Новый диалог** - Начать новый разговор
- 🧠 **Память** - Посмотреть сохраненные сообщения
- ❌ **Сбросить память** - Очистить всю память

## 🚨 Решение проблем

### Бот не отвечает
- Проверьте что TG_TOKEN и FIREWORKS_KEY установлены правильно
- Убедитесь в наличии интернета
- Проверьте лимиты Fireworks API

### Ошибка при подключении к API
- Проверьте валидность FIREWORKS_KEY
- Убедитесь что у вас есть баланс на Fireworks

### Память не сохраняется
- Проверьте права доступа к папке проекта
- Убедитесь что есть место на диске
